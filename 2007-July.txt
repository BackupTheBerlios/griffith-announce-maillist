Griffith version 0.9.3 has been released today.

Changelog:

* New configuration file format (human readable) - please note that
data from old config file is not used
* Updated movie plugins: Filmweb, WP, FDb, IMDb, 7arte.net, Cineteka,
Kino.de, DVD-Palace, E-Pipoca, Stopklatka
* New movie plugins: FilmDb, Zelluloid, IMDb-de
* New translations:
  + Dutch (thanks to Marcel Dijkstra)
  + Danish (thanks to Joe Dalton)
  + Greek (thanka to Ioannis Koniaris)
  + Simplified Chinese (thanks to kempson)
* Updated translations: German, Spanish, French, Polish and Portuguese
Brazilian (from Rosetta)
* Added new alpha export plugin for iPods
* Added Microsoft SQL support
* Lots of bugs fixed

As usual, grab it from http://griffith.berlios.de/

Have fun!
