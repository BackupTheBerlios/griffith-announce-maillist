Griffith version 0.9.6 has been released.

This is mainly a bug fix release. This versions contains local copy of
SQLAlchemy v0.3.11. Griffith 0.10 which will work with SQLAlchemy v0.4.x
is not ready yet (see trunk).

Changes:
* Updated movie plugins: AniDb, FilmeVonAZ, Kino.de, OFDb, Stopklatka,
  Zelluloid
* Translation changes:
  + new: Estonian, Hungarian, Indonesian, Korean, Occitan (post 1500),
    Pashto, Slovak
  + updated: almost all :-)
* Quick filter for volumes
* Added columns genre, seen, runtime and year to the mainlist

Please see ChangeLog for more details.
